# real-estate-price-analytics

📌 Aim

The aim of this project is to perform an in-depth exploratory data analysis on a real-world housing dataset to uncover meaningful patterns, relationships, and key factors influencing house prices. By analyzing structural features such as living area, number of bedrooms and bathrooms, location, and property condition, this study seeks to identify the primary drivers of price variation. The analysis also focuses on understanding data distributions, detecting outliers, evaluating correlations, and transforming variables to improve interpretability. Ultimately, the goal is to build a strong analytical foundation that can support future predictive modeling and data-driven decision-making in the real estate domain.

## Project Overview

This project conducts a comprehensive EDA on a dataset containing over 21,000 house records with 21 features. The workflow includes:

Understanding dataset structure and data types
Identifying missing values (none found)
Performing descriptive statistical analysis
Exploring unique values across features
Feature engineering (extracting year and month from date)
Analyzing numerical, discrete, and continuous variables
Visualizing relationships between features and house prices
Detecting outliers using boxplots
Applying log transformations for better distribution analysis
Evaluating feature correlations using heatmaps and pair plots

## Key Insights
House prices tend to increase over time, showing a strong temporal trend.
Features like sqft_living, grade, and bathrooms significantly impact pricing.
Waterfront properties and houses with better views command higher prices.
Optimal ranges exist for features like bedrooms and floors, beyond which price gains plateau.
Strong correlations exist between living space-related features and price.
